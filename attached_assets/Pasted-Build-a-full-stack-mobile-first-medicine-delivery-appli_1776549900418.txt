Build a full-stack mobile-first medicine delivery application named **"Medirush"**, similar to Blinkit (10-minute delivery concept). The project should be production-ready, cleanly structured, and easily deployable. I will export this project to GitHub, so maintain proper folder structure, README, and environment configs.

## 🔧 Tech Stack

* Frontend: React Native (Expo) OR Next.js (mobile responsive if RN not possible)
* Backend: Node.js + Express
* Database: MongoDB (Mongoose)
* Authentication: JWT-based auth
* File Upload: Cloudinary or local storage (for prescription uploads)
